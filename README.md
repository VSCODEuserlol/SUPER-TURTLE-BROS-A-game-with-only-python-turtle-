# SUPER-TURTLE-BROS-A-game-with-only-python-turtle-
======Version 1.0.0=======

## What is Super turtle bros?

-A game made with ONLY python turtle

-A fun game to play with your friends

-Much, much, more!

## Here's how to Play:

Player 1: WASD [W:up A:left D:right S:down]

Attack: F

Super: G

[If boss] Special: E

Player 2: Arrow Keys

Attack:M

Super:G

[If boss] Special: o
